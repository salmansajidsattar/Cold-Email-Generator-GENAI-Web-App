
# Generative AI Cold Email Sender

The Generative Cold Email Sender is a cutting-edge application designed to simplify and automate the process of crafting personalized cold emails to recruiters. This app takes a job web link, fetches the job posting data, and uses Groq API LLM to generate a compelling email tailored for the specific job role.

## Key Features

**Automatic Job Data Extraction:** Enter a job link, and the app will fetch all relevant job posting details.

**Personalized Email Generation:** Utilizes Groq API LLM to create custom emails based on the job data.

**Efficient Backend:** Built with Flask to handle backend processes smoothly.

**Interactive Frontend:** Designed using HTML, CSS, and JavaScript for a user-friendly and responsive interface.

**Seamless User Experience:** Simple and intuitive design to ensure ease of use.

## Tech Stack
**Backend**: Pyton-Flask

**Frontend**: HTML, Bootstrap, JavaScript

**AI Integration**: Groq API LLM

**Vector DB**: Chroma database

## How to Use


**Enter Job Link:** Input the job listing URL into the provided field.

**Fetch Job Data:** Click on the "Fetch Data" button to retrieve job details.

**Generate Email:** The app will craft a personalized email using the job data.

**Review & Send:** Review the generated email content and send it directly to the recruiter.
